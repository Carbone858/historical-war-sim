# 🌍 Historical War Simulation Platform
Multi-platform historical war simulator with real-time maps, AI "what-if" scenarios, and verified data.

## 🚀 Run Locally
\`\`\`bash
cp .env.example .env
# Edit .env & add your MAPBOX_TOKEN
docker compose up -d
\`\`\`
- Frontend: `http://localhost:3000/b-001`
- Backend API: `http://localhost:8000/docs`
